## Windows 访问控制

- 输入密码后，系统将密码保存到缓存中，以防重复输入
- 为用户创建

- whoami
- SID
  - S
  - SID版本号
  - 颁发机构 ID
  - 域或本地 ID
  - 账户和组 ID

- 打开一个进程 Explorer.exe，每个用户有每个用户的 Token，非管理员用户之间不能相互访问
